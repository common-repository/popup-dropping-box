== Description ==

This popup plugin drops your text in the popup window. So that the text on the popup window in your page gets the attention to your users. You can use this plugin to show your important announcement or advertisement.

Check official website for live demo [http://www.gopiplus.com/work/2019/12/25/popup-dropping-box-wordpress-plugin/](http://www.gopiplus.com/work/2019/12/25/popup-dropping-box-wordpress-plugin/)

* Live Demo : [Live Demo](http://www.gopiplus.com/work/2019/12/25/popup-dropping-box-wordpress-plugin/)	
* More Information : [More Information](http://www.gopiplus.com/work/2019/12/25/popup-dropping-box-wordpress-plugin/)				
* Plugin : [Plugin URI](http://www.gopiplus.com/work/2019/12/25/popup-dropping-box-wordpress-plugin/)		

**Plugin features**

* Supports all major browsers.
* Responsive width.
* Easily position the popup window.
* Specify whether the popup box should automatically disappear after X seconds .
* Support to show multiple popup boxes on the page.
* Option to control frequency (always, once per browser session)

== Installation ==

= Install Plugin using WordPress Plugin Search =

Go to your WordPress admin area and click on Plugins >> Add New. Search Popup dropping box plugin using search option. Find the plugin and click Install Now link. Finally click activate plugin link to activate the plugin.

[Popup dropping box](http://www.gopiplus.com/work/2019/12/25/popup-dropping-box-wordpress-plugin/ "Popup dropping box WordPress plugin")

= Install Plugin using the WordPress Admin Plugin Upload =

Download the plugin popup-dropping-box.zip from [download](http://www.gopiplus.com/work/2019/12/25/popup-dropping-box-wordpress-plugin/ "popup-dropping-box.zip") location. Go to Plugins >> Add New in your WordPress admin. Select upload plugin button. Upload the available popup-dropping-box.zip file and click Install Now. Finally click activate plugin link to activate the plugin.
